Enunciado
=========

Rotar los elementos 270º.
Intercambiar las posiciones de las cajas 1 y 2, 3 y 4.
Cambiar el tamaño de 100px a 50px.
Cambiar el color de fondo de azul a rojo.
Cambiar el color del texto de negro a amarillo.
Cambiar el tamaño de letra de 20px a 18px

Notas:
======

La animación no se coloca en el evento hover. Se indica en el estado normal.
De lo contrario no se dará animación al perder hover sobre el elemento.

La sobreescritura de transform no se suma, sino que se suistituye, lo que implica repetición de código.
